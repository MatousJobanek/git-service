# git-service
This service provides functionality for communications with git providers in DevConsole

### Pre-requisites
- [dep][dep_tool] version v0.5.0+.
- [git][git_tool]
- [go][go_tool] version v1.10+.
- [docker][docker_tool] version 17.03+.

### To Download Dependencies
```
make deps
```

### Build 
```
make build
```

### Clean 
```
make clean
```
